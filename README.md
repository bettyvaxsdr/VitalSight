# VitalSight

VitalSight е система, която се състои от уебсайт, камера и устройство, измерващо сърдечен ритъм и телесна температура на възрастен човек в реално време.

Ако тези данни са критични се изпраща сигнал към камерата за потвърждение. Тя следи за липсата на движение на човека.

Данните се визуализират в уебсайта и при опасни отклонения е възможна бърза реакция от близките на човека.
